SECURITY CONTROL MAPPING
========================

LLMGuard Controls:
- Input Scanning: Prompt injection detection
- Output Sanitization: Sensitive data redaction
- Banned Substrings: Specific phrase blocking
- Toxicity Detection: Harmful content detection
- Language Detection: Non-English content filtering

OPA Controls:
- Role-Based Access: User < Manager < Admin hierarchy
- Field-Level Security: PII field blocking
- Time-Range Restrictions: Historical data limits
- Export Controls: Business purpose validation
- Domain Validation: Internal vs external email domains
